# 🚀 Flagship Work

## 📈 Stock-Agent: Multi-Agent Portfolio Governance Engine

Advanced multi-agent AI decision system that formalizes high-conviction,
high-beta investing into a structured governance framework.

Stock-Agent functions as a conviction governance system with
deterministic aggregation, provider-resilient market data ingestion, and
executive-readable reporting.

### 🧠 Multi-Agent Graph Architecture

Each position is evaluated by specialized agents:

-   Thesis Integrity Agent\
-   Risk Exposure Agent\
-   Sentiment Agent\
-   Skeptic (Adversarial) Agent\
-   Bias Detection Agent\
-   Consistency Agent\
-   Judge Aggregation Agent

Agents operate independently, and outputs are aggregated
deterministically to ensure repeatable, explainable results.

### ⚖ Counterbalance Engine

Effective Buy Score:

EffBuy = Raw Buy − (Risk Weight × Risk + Skeptic Pressure + Agent
Conflict)

This prevents conviction inflation, bias reinforcement, and fragility
blindness.

### 📊 Quant Overlay Layer

Quant is used for structural awareness --- not prediction.

Includes:

-   Beta vs SPY\
-   Annualized volatility\
-   12--1 momentum\
-   Correlation clustering\
-   HHI concentration analysis\
-   Historical VaR & Expected Shortfall\
-   Monte Carlo drawdown simulation\
-   Liquidity classification\
-   Volume regime detection

Flow overlay includes:

-   5-day momentum\
-   20-day momentum\
-   20-day annualized volatility\
-   Volume ratio\
-   Liquidity flags

### 🔄 Dual-Provider Market Data Architecture

Primary: Yahoo Finance (yfinance)\
Fallback: Stooq daily CSV endpoint

Runtime behavior:

1.  Check cache\
2.  Attempt Yahoo fetch\
3.  If empty/blocked → fallback to Stooq\
4.  If both fail → mark data unavailable explicitly

Each ticker includes a Data Coverage Indicator to prevent silent data
gaps.

Repo: https://github.com/Bellbotics/stock-agent
